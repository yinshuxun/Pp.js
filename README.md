#pP.js
    这是一个图片预览插件(初版)
    [demo演示](https://yinshuxun.github.io/pPjs/index.html)

##使用 
```
使用如下:
new Pp({
                        id: "picc",
                        zIndex: "10000",
                        picList: picList,
                        currIndex: startIndex
                    });

```
